# Portfolio officiel de Manassé Joseph MOUSSESSA

Ce projet présente un portfolio personnel et officiel réalisé en HTML, CSS et Tailwind CSS.

## Fichiers principaux

- `index.html` : page principale du portfolio.
- `style.css` : styles complémentaires pour le design.
- `manasse.jpg` : photo personnelle utilisée dans la page.
- `Comprendre la trigonométrie en 15 minutes.png` : image de démonstration du contenu pédagogique.

## Utilisation

1. Ouvrez le fichier `index.html` dans votre navigateur.
2. Modifiez les informations personnelles directement dans `index.html` si nécessaire.
3. Changez les couleurs ou le style dans `style.css`.
4. Le lien vers la chaîne YouTube pointant vers `https://www.youtube.com/@Laperfectionschool` est déjà configuré.

## Détails

- Le site utilise la version CDN de Tailwind CSS via `<script src="https://cdn.tailwindcss.com"></script>`.
- Le code est rédigé en français et comprend des commentaires pour faciliter les modifications.
- Un mode sombre élégant est appliqué avec des arrière-plans et des cartes modernes.

## Remarque

Pour ajouter d'autres éléments ou sections, copiez une carte existante et modifiez son contenu.
